# Die wichtigsten LVGL 9.x Widgets die auf einem ESP32-S3 laufen

| Widgetname                        | Type             | URL                                                           | Beispiel                                                                                                                                                                     | Information                                                    | Touch nötig? | Ohne Touch nutzbar? Wie                                                  |
| --------------------------------- | ---------------- | ------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------- | ------------ | ------------------------------------------------------------------------ |
| Label (`lv_label`)                | Label            | https://docs.lvgl.io/master/details/widgets/label.html        | ```c\nlv_obj_t* l = lv_label_create(lv_screen_active());\nlv_label_set_text(l, "Hello");\nlv_obj_center(l);\n```                                                             | Leichtgewichtig, Auto-Wrap, Recolor. Ideal für Statusanzeigen. | Nein         | Nicht interaktiv; Fokus i. d. R. unnötig.                                |
| Button (`lv_button`)              | Input            | https://docs.lvgl.io/master/details/widgets/button.html       | ```c\nlv_obj_t* btn = lv_button_create(lv_screen_active());\nlv_obj_add_event_cb(btn, cb, LV_EVENT_CLICKED, NULL);\n```                                                      | Klick-Button als Basis für Aktionen; enthält oft Label/Icon.   | Nein         | Fokus mit Encoder/Tasten, **ENTER** triggert Klick.                      |
| Button Matrix (`lv_buttonmatrix`) | Input            | https://docs.lvgl.io/master/details/widgets/buttonmatrix.html | ```c\nlv_obj_t* bm = lv_buttonmatrix_create(lv_screen_active());\nstatic const char* map[] = {"1","2","3","\n","4","5","6","",NULL};\nlv_buttonmatrix_set_map(bm, map);\n``` | Soft-Tastenraster; ideal als Numpad/Keypad.                    | Nein         | Fokus, dann mit LEFT/RIGHT/UP/DOWN navigieren, **ENTER** aktivieren.     |
| Switch (`lv_switch`)              | Input            | https://docs.lvgl.io/master/details/widgets/switch.html       | ```c\nlv_obj_t* sw = lv_switch_create(lv_screen_active());\n```                                                                                                              | On/Off-Schalter mit Animation.                                 | Nein         | **ENTER** toggelt den Zustand.                                           |
| Checkbox (`lv_checkbox`)          | Input            | https://docs.lvgl.io/master/details/widgets/checkbox.html     | ```c\nlv_obj_t* cb = lv_checkbox_create(lv_screen_active());\nlv_checkbox_set_text(cb, "Enable");\n```                                                                       | Checkbox inkl. Text.                                           | Nein         | **ENTER** toggelt; mit UP/DOWN Fokus wechseln.                           |
| Slider (`lv_slider`)              | Input            | https://docs.lvgl.io/master/details/widgets/slider.html       | ```c\nlv_obj_t* s = lv_slider_create(lv_screen_active());\n```                                                                                                               | Wertewahl linear; optional mit Label.                          | Nein         | Fokus → mit LEFT/RIGHT (oder UP/DOWN) den Wert ändern.                   |
| Spinbox (`lv_spinbox`)            | Input            | https://docs.lvgl.io/master/details/widgets/spinbox.html      | ```c\nlv_obj_t* sb = lv_spinbox_create(lv_screen_active());\n```                                                                                                             | Zahlenfeld mit Pfeilen; encoder-freundlich.                    | Nein         | Fokus → drehen erhöht/vermindert; **ENTER** wechselt Stelle (optional).  |
| Text Area (`lv_textarea`)         | Input            | https://docs.lvgl.io/master/details/widgets/textarea.html     | ```c\nlv_obj_t* ta = lv_textarea_create(lv_screen_active());\n```                                                                                                            | Mehrzeilige Eingabe; Password/Masking möglich.                 | Nein         | Mit **lv_keyboard** oder eigenen Key-Events befüllen.                    |
| Keyboard (`lv_keyboard`)          | Input            | https://docs.lvgl.io/master/details/widgets/keyboard.html     | ```c\nlv_obj_t* kb = lv_keyboard_create(lv_screen_active());\n```                                                                                                            | Bildschirmtastatur, koppelt an Text Area.                      | Nein         | Fokus per Encoder; Tasten mit **ENTER** drücken.                         |
| Dropdown (`lv_dropdown`)          | Input            | https://docs.lvgl.io/master/details/widgets/dropdown.html     | ```c\nlv_obj_t* dd = lv_dropdown_create(lv_screen_active());\nlv_dropdown_set_options(dd, "Red\nGreen\nBlue");\n```                                                          | Klappauswahl mit Filter/Dir-Modus.                             | Nein         | **ENTER** öffnet, UP/DOWN wählen, **ENTER** bestätigt, **ESC** schließt. |
| Roller (`lv_roller`)              | Input            | https://docs.lvgl.io/master/details/widgets/roller.html       | ```c\nlv_obj_t* r = lv_roller_create(lv_screen_active());\n```                                                                                                               | „Picker-Rad“; sehr MCU-freundlich.                             | Nein         | Dreh/UP/DOWN wählen; **ENTER** bestätigt.                                |
| Arc (`lv_arc`)                    | Grafik           | https://docs.lvgl.io/master/details/widgets/arc.html          | ```c\nlv_obj_t* arc = lv_arc_create(lv_screen_active());\n```                                                                                                                | Halb/Vollkreis; als Progress/Knob.                             | Nein         | Optional interaktiv: Wert per Keys ändern.                               |
| Scale (`lv_scale`)                | Grafik           | https://docs.lvgl.io/master/details/widgets/scale.html        | ```c\nlv_obj_t* sc = lv_scale_create(lv_screen_active());\n```                                                                                                               | Skala mit Ticks/Labels; v9-Style statt „Gauge“.                | Nein         | Anzeige; interaktiv in Kombi mit Arc/Knob.                               |
| Arc Label (`lv_arclabel`)         | Grafik/Label     | https://docs.lvgl.io/master/details/widgets/arclabel.html     | ```c\nlv_obj_t* al = lv_arclabel_create(lv_screen_active());\n```                                                                                                            | Text entlang eines Kreisbogens.                                | Nein         | Anzeige (nicht interaktiv).                                              |
| Chart (`lv_chart`)                | Grafik           | https://docs.lvgl.io/master/details/widgets/chart.html        | ```c\nlv_obj_t* ch = lv_chart_create(lv_screen_active());\n```                                                                                                               | Linien/Balken, Achsen, Cursors.                                | Nein         | Navigation optional über Fokus + Keys (z. B. Cursor).                    |
| Bar (`lv_bar`)                    | Grafik           | https://docs.lvgl.io/master/details/widgets/bar.html          | ```c\nlv_obj_t* b = lv_bar_create(lv_screen_active());\n```                                                                                                                  | Progressbar einfach/vertical/horizontal.                       | Nein         | Anzeige; Werte per API setzen, Keys optional.                            |
| Line (`lv_line`)                  | Grafik           | https://docs.lvgl.io/master/details/widgets/line.html         | ```c\nlv_obj_t* ln = lv_line_create(lv_screen_active());\n```                                                                                                                | Linien/Polylinien, sehr leicht.                                | Nein         | Anzeige.                                                                 |
| Canvas (`lv_canvas`)              | Grafik           | https://docs.lvgl.io/master/details/widgets/canvas.html       | ```c\nlv_obj_t* cv = lv_canvas_create(lv_screen_active());\n```                                                                                                              | Pixel-genaues Zeichnen in RAM/PSRAM.                           | Nein         | Anzeige; Interaktion selbst definieren.                                  |
| Image (`lv_image`)                | Grafik           | https://docs.lvgl.io/master/details/widgets/image.html        | ```c\nlv_obj_t* img = lv_image_create(lv_screen_active());\n```                                                                                                              | Bilder (RAW/BMP/PNG/JPEG via Decoder).                         | Nein         | Anzeige.                                                                 |
| Image Button (`lv_imagebutton`)   | Input/Grafik     | https://docs.lvgl.io/master/details/widgets/imagebutton.html  | ```c\nlv_obj_t* ib = lv_imagebutton_create(lv_screen_active());\n```                                                                                                         | Bild-Buttons mit State-Grafiken.                               | Nein         | Fokus + **ENTER** zum Auslösen.                                          |
| List (`lv_list`)                  | Container/Navi   | https://docs.lvgl.io/master/details/widgets/list.html         | ```c\nlv_obj_t* list = lv_list_create(lv_screen_active());\n```                                                                                                              | Scrollbare Liste; Einträge sind Buttons.                       | Nein         | Fokus, dann UP/DOWN; **ENTER** aktiviert Eintrag.                        |
| Menu (`lv_menu`)                  | Container/Navi   | https://docs.lvgl.io/master/details/widgets/menu.html         | ```c\nlv_obj_t* m = lv_menu_create(lv_screen_active());\n```                                                                                                                 | Hierarchisches Menü mit Seiten/Abschnitten.                    | Nein         | Navigieren per Keys, **ENTER** wählen, **ESC** zurück.                   |
| Tabview (`lv_tabview`)            | Container/Navi   | https://docs.lvgl.io/master/details/widgets/tabview.html      | ```c\nlv_obj_t* tv = lv_tabview_create(lv_screen_active(), LV_DIR_TOP, 32);\n```                                                                                             | Registerkarten-Navigation.                                     | Nein         | Fokus auf Tabs; LEFT/RIGHT oder UP/DOWN wechselt Tabs.                   |
| Table (`lv_table`)                | Daten/Label      | https://docs.lvgl.io/master/details/widgets/table.html        | ```c\nlv_obj_t* t = lv_table_create(lv_screen_active());\n```                                                                                                                | Tabellenansicht mit Zellenformatierung.                        | Nein         | Fokus, dann mit Keys Zellen selektieren (optional).                      |
| Message Box (`lv_msgbox`)         | Dialog           | https://docs.lvgl.io/master/details/widgets/msgbox.html       | ```c\nlv_obj_t* mb = lv_msgbox_create(NULL, "Title", "Text", NULL, true);\n```                                                                                               | Modaler Dialog mit Buttons.                                    | Nein         | Fokus landet auf Buttons; **ENTER** bestätigt.                           |
| Window (`lv_win`)                 | Container/Dialog | https://docs.lvgl.io/master/details/widgets/win.html          | ```c\nlv_obj_t* w = lv_win_create(lv_screen_active(), 30);\n```                                                                                                              | Fensterrahmen mit Header/Content.                              | Nein         | Fokus auf Header-Buttons/Content steuerbar.                              |
| Tileview (`lv_tileview`)          | Container/Navi   | https://docs.lvgl.io/master/details/widgets/tileview.html     | ```c\nlv_obj_t* tv = lv_tileview_create(lv_screen_active());\n```                                                                                                            | Wischbare Kacheln/Seiten.                                      | Nein         | Per Keys zwischen Tiles wechseln.                                        |
| LED (`lv_led`)                    | Grafik/Status    | https://docs.lvgl.io/master/details/widgets/led.html          | ```c\nlv_obj_t* led = lv_led_create(lv_screen_active());\n```                                                                                                                | Farbiger Statuspunkt mit Helligkeit.                           | Nein         | Anzeige; Zustand per API setzen.                                         |

# Eingabesysteme

| Name                            | Type                  | Info                                                                                                                            | Beispiel                                                                                                                                                                             | URL                                                                  |
| ------------------------------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------- |
| Encoder (Rotary)                | Digital / Incremental | Drehgeber mit optionalem Drucktaster. Wird über `LV_INDEV_TYPE_ENCODER` eingebunden. Ideal zur Navigation ohne Touchscreen.     | ```c\nstatic lv_indev_t* indev_enc;\nindev_enc = lv_indev_create();\nlv_indev_set_type(indev_enc, LV_INDEV_TYPE_ENCODER);\nlv_indev_set_read_cb(indev_enc, my_encoder_read_cb);\n``` | https://docs.lvgl.io/master/overview/indev.html#encoder              |
| GPIO Button                     | Digital               | Einfache Taster, die Aktionen oder Navigation triggern. Über `LV_INDEV_TYPE_KEYPAD` oder benutzerdefinierte Events eingebunden. | ```c\nstatic lv_indev_t* indev_btn;\nindev_btn = lv_indev_create();\nlv_indev_set_type(indev_btn, LV_INDEV_TYPE_KEYPAD);\nlv_indev_set_read_cb(indev_btn, my_button_read_cb);\n```   | https://docs.lvgl.io/master/overview/indev.html#keypad               |
| Touchscreen (SPI/I2C)           | Touch / Analog        | Resistiver oder kapazitiver Touchcontroller (z. B. XPT2046, FT6206). Wird als `LV_INDEV_TYPE_POINTER` registriert.              | ```c\nstatic lv_indev_t* indev_touch;\nlv_indev_set_type(indev_touch, LV_INDEV_TYPE_POINTER);\n```                                                                                   | https://docs.lvgl.io/master/overview/indev.html#pointer              |
| Mouse / Touchpad (USB)          | Pointer               | Externer Mauszeiger über USB Host oder BLE. Simuliert Touch-Ereignisse (`LV_INDEV_TYPE_POINTER`).                               | ```c\nlv_indev_t* indev_mouse = lv_indev_create();\nlv_indev_set_type(indev_mouse, LV_INDEV_TYPE_POINTER);\n```                                                                      | https://docs.lvgl.io/master/overview/indev.html                      |
| Keypad / Matrix Keyboard        | Digital / Matrix      | Mehrere Buttons (z. B. 4x4 Keypad) mit dedizierten GPIOs oder Multiplex. Ideal für Menüs.                                       | ```c\n// LV_KEY_ Codes senden in read_cb\n```                                                                                                                                        | https://docs.lvgl.io/master/overview/indev.html#keypad               |
| ADC / Analog Joystick           | Analog                | 2D-Joystick mit X/Y Analogwerten. In `read_cb` in Koordinaten übersetzt.                                                        | ```c\nlv_indev_t* indev_js = lv_indev_create();\nlv_indev_set_type(indev_js, LV_INDEV_TYPE_POINTER);\n```                                                                            | https://docs.lvgl.io/master/overview/indev.html#pointer              |
| External Keyboard (USB/BLE)     | Keyboard / Digital    | Über HID oder UART eingebundene Tastatur. Liefert Key-Codes (z. B. LV_KEY_LEFT, LV_KEY_ENTER).                                  | ```c\nlv_indev_t* indev_key;\nlv_indev_set_type(indev_key, LV_INDEV_TYPE_KEYPAD);\n```                                                                                               | https://docs.lvgl.io/master/overview/indev.html#keypad               |
| IR Remote                       | Digital               | IR-Empfänger (z. B. NEC-Protokoll) → Keycodes konvertiert. Praktisch für Fernbedienungen.                                       | ```c\n// In read_cb je nach IR-Code LV_KEY_* senden\n```                                                                                                                             | -                                                                    |
| UART Command Input              | Serial / Virtual      | Serielle Eingabe (z. B. CLI, Remote Control), sendet virtuelle LVGL-Events.                                                     | ```c\nif(cmd == "UP") lv_event_send(target, LV_EVENT_KEY, LV_KEY_UP);\n```                                                                                                           | -                                                                    |
| BLE HID Device                  | Wireless              | BLE-HID Tastatur, Maus oder Encoder; funktioniert wie USB-Eingabe.                                                              | ```c\n// BLE Callback sendet LV_KEY Codes\n```                                                                                                                                       | https://docs.lvgl.io/master/overview/indev.html                      |
| Custom Input (Sensor / Gesture) | Virtuell              | Beliebige benutzerdefinierte Sensoren (z. B. Gyro, Gesten, Touchpad) → LVGL Events.                                             | ```c\nlv_indev_set_read_cb(indev_custom, my_read_cb);\n```                                                                                                                           | https://docs.lvgl.io/master/overview/indev.html#custom-input-devices |
